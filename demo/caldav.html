timeseries.js — CalDAV

calendar events as spans, via the caldav source and gantt renderer

← Main demo

No server is configured, so this page parses the static fixtures in demo/fixtures/ and shifts them onto the current week. To talk to a real server, set CALDAV at the top of the page script — the caldav source then discovers the calendars and refetches as you pan. A cross-origin server must answer the OPTIONS preflight without demanding authentication, listing PROPFIND and REPORT in Access-Control-Allow-Methods and Authorization, Depth, Content-Type in Access-Control-Allow-Headers. Requests go out with credentials: 'omit', so a wildcard Access-Control-Allow-Origin is enough. Most public CalDAV servers fail the preflight (they answer it with 401) — for those, set proxy and forward the request same-origin.
